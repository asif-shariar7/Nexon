# Nexon AI

Nexon AI is a modern, responsive AI assistant web application that delivers fast, intelligent, and real-time conversations using advanced AI models.


## **Video Demo:**  
https://drive.google.com/file/d/1_sWyuSbN-8vRuFD9han_0C_hmiIAQRZA/view?usp=sharing


## Features

- Fast AI-powered responses
- Smart contextual understanding
- Fully responsive design (Desktop, Tablet, Mobile)
- Modern dark futuristic UI
- Real-time chat interface
- Typing indicator animation
- Smooth scrolling navigation
- Clean and structured codebase

## Built With

- HTML5
- CSS3 (Flexbox & Responsive Design)
- Vanilla JavaScript (ES6)
- Google Gemini API

> **Note:** You must add your own Google Gemini API key inside the `API_KEY` variable in the `js/main.js` file. Without your API key, the chat functionality will not work.
